# OC — Portfolio

Personal portfolio site. Built with HTML, Tailwind CSS, and vanilla JavaScript.

## Stack

- HTML5
- Tailwind CSS (Play CDN)
- Vanilla JavaScript
- Font Awesome 6
- Google Fonts (Fraunces + DM Sans)

## Structure
```
portfolio/
├── index.html   — markup and structure
├── styles.css   — custom CSS and animations
├── script.js    — scroll reveal, mobile menu, contact form
└── README.md
```

## Contact Form

Uses [Formspree](https://formspree.io). Replace the `action` URL in
`index.html` with your own endpoint before deploying.
